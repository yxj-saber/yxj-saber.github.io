2. pycharm创建Django项目

  • pycharm虚拟环境：

    pip check – 检查依赖库是否有冲突

    pip freeze – 查看已安装依赖库以及版本

    pip freeze > requirements.txt – 将安装的依赖库环境重定向到requirements的文本文件中

    pip install -r requirements.txt – 根据requirements文件重新安装依赖库环境

  • 通过PyCharm创建python项目 —> hellodjango

    (1) 在Terminal的虚拟环境中安装依赖 —> pip install django pymysql django-redis djangorestframework pillow

    (2) pip check

    (3) pip freeze > requirements.txt

    (4) 创建Django项目 —> django-admin startproject hellojango .

    (5) 运行项目 —> python manage.py runserver

    ​ http://localhost:8000

    ​ http://127.0.0.1:8000

    (6) python manage.py runserver 0.0.0.0:80

    ​ http://localhost

    ​ http://127.0.0.1

    ​ http://39.108.251.113

    (7) 修改配置文件 —> hellodjango/setting.py

    ​ LANGUAGE_CODE = ‘zh-hans’ –> 设置中文

    ​ TIME_ZONE = ‘Asia/Chongqing’ –> 设置时区

    ​ DATABASES = { –> 设置数据库连接

​ }

​ 修改__init__.py文件

​ (8) 生成Django自带的数据表

​ python manage.py migrate

​ (9) 创建后台管理账号

​ python manage.py createsuperuser
